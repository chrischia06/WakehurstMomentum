# Installation
Download this repository, either using ``git clone https://github.com/chrischia06/WakehurstMomentum.git`` or direct download

run ``pip3 -r requirements.txt`` to install dependencies, or using your dependency manager

# Usage

There are two notebooks to backtest strats: 

+ WakehurstMomentum - uses machine learning to forecast short term direction/price/rate/log_returns based on past prices & returns

+ Wakehurst_Beta_Rv - determine cheapness/expensiveness of an asset relative to other assets or factors
